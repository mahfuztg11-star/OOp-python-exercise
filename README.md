 OOp-python-exercise
Python OOP Practice Exercises

This repository contains beginner-friendly Object-Oriented Programming (OOP) exercises in Python. The exercises are designed to help learners understand the core concepts of classes, objects, constructors, and destructors.

Topics Covered

* Classes
* Objects
* Constructors (`__init__`)
* Destructors (`__del__`)
* Object Attributes

 Exercises

 1. Student Class

A simple class that creates student objects with different names and displays messages when objects are created and destroyed.

 2. Car Class

A class representing cars with attributes such as brand and model. Constructor and destructor messages are displayed for each object.
 3. Book Class

A class that stores book information including title and author and demonstrates object lifecycle management.

 4. Account Class

A banking account example that stores account numbers and balances while showing constructor and destructor functionality.

 5. Mobile Class

A class representing mobile phones with brand and price attributes. Multiple objects are created to practice OOP concepts.

 Learning Objectives

By completing these exercises, learners will be able to:

* Create and use Python classes.
* Instantiate objects from classes.
* Understand how constructors work.
* Understand the purpose of destructors.
* Store and access object attributes.
* Build a strong foundation for advanced OOP concepts.
